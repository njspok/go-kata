# Average consensus algorithm

Распределённый алгоритм усреднения (average consensus) на графе узлов. 
Каждый узел хранит число и периодически обменивается значениями с
соседями, постепенно сходясь к общему среднему.

Особенности
- не нужен центральный узел и значение хранится на каждом узле
- узлы могут иметь связь только с ограниченным числом других узлов в графе
- реализует согласованность по итогу
